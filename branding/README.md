# Prism Branding

Brand assets and guidelines for the Prism blockchain project.

## Contents

| File | Description |
|------|-------------|
| `logo.svg` | Primary hexagonal logo mark (purple-to-green gradient, "SC" letterform, 200x200 viewport) |
| `brand-guidelines.md` | Full brand specification including colors, typography, components, and voice |

## Brand Summary

- **Name**: Prism
- **Token**: PRISM
- **Tagline**: High-Performance Blockchain, Redefined
- **Primary Colors**: Purple `#9945FF`, Green `#14F195`
- **Gradient**: `linear-gradient(135deg, #9945FF 0%, #14F195 100%)`
- **Background**: Dark `#0F0B2E`
- **UI Font**: Inter (400/500/600/700)
- **Code Font**: JetBrains Mono (400)

## Usage

The `brand-guidelines.md` file contains the full specification: color palette (primary, secondary, semantic), typography scale, spacing tokens, border radii, button/card CSS, logo usage rules, and writing style guidelines.
